Busca una ruta entre dos conceptos, módulos, specs o archivos usando Graphify.

Argumentos recibidos:

```text
$ARGUMENTS
```

Ejecuta `graphify path` con los dos conceptos indicados por el usuario. Si los argumentos no separan claramente origen y destino, pide una aclaración breve.
Usa este comando para preguntas del tipo “cómo se conecta A con B”, “qué relación hay entre una spec y un módulo”, o “qué toca este cambio”.
